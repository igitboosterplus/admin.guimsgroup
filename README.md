# GUIMS GROUP — HR Hub

Plateforme de gestion des ressources humaines pour GUIMS GROUP.

## Technologies

- React 18 + TypeScript
- Vite
- Supabase (Auth, PostgreSQL, RLS)
- Shadcn/ui + Tailwind CSS
- jsPDF (export PDF)
- Recharts (graphiques)

## Fonctionnalités

- Gestion des employés (CRUD, départements, postes, archivage)
- Pointage et présence (WiFi, géolocalisation)
- Tableau de bord avec statistiques et graphiques
- Rapports mensuels de salaire (tableau, CSV, PDF)
- Demandes de congés (payés / non payés)
- Gestion des tâches et missions
- Système de permissions par rôle
- Approbation des nouveaux comptes
- Profil employé

## Installation

```sh
npm install
npm run dev
```
